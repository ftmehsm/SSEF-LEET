# Week 01 — My Solution / راه‌حل من

**Author / نویسنده:** ftmehsm  
**Language / زبان:** JavaScript  
**Approach / رویکرد:** Modulo checks + loop / بررسی باقی‌مانده و حلقه

---

## Idea / ایده

**EN:**
1. Loop from `1` to `n`
2. For each number, use `%` to check divisibility by 3 and 5
3. Push `"fizz"`, `"Buzz"`, `"Fizz Buzz"`, or the number itself into a result array

<div dir="rtl">

**FA:**
1. از `1` تا `n` حلقه می‌زنم
2. برای هر عدد با `%` بخش‌پذیری بر ۳ و ۵ را چک می‌کنم
3. `"fizz"`، `"Buzz"`، `"Fizz Buzz"` یا خود عدد را داخل آرایه نتیجه می‌گذارم

</div>

## Complexity / پیچیدگی
